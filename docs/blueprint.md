# **App Name**: CareerCompass Kenya

## Core Features:

- MI Diagnostic & Scoring Engine: A 45-question diagnostic engine that maps student interests and passions to Howard Gardner's nine intelligence types using an optimized scoring algorithm.
- Intelligence Visualization Dashboard: A results portal that visualizes dominant and co-dominant intelligence profiles using interactive radar and bar charts for student self-discovery.
- CBE Pathway & Subject Optimizer: A logic-driven classification system that recommends official Kenyan CBE pathways and validates subject combinations against the 11-subject rule.
- ML Career & Hobby Predictor: A machine learning engine that correlates intelligence combinations and personal hobbies with Kenyan career opportunities, salary ranges, and market demand.
- Automated Study & Growth Planner: Generates personalized 7-day academic timetables and multi-year career development plans tailored to a student's specific intelligence profile and strengths.
- KUCCPS & Mentorship Hub: Provides university cluster point estimation and a directory of official links to Kenyan education bodies including KICD, KUCCPS, and mentorship programs.
- Admin Analytics & PDF Utility: An administrative portal that allows schools to track student distributions across pathways and export high-fidelity downloadable career guidance reports.
- Integrated Feedback Portal: A feedback system where students can rate the application and submit qualitative suggestions to ensure the platform remains effective and student-friendly.

## Style Guidelines:

- Primary color: Deep Indigo-Blue (#2866BD) to establish trust, institutional authority, and professional clarity.
- Background color: Ultra-light slate (#F7F8FA) for a clean, academic reading environment that minimizes eye strain.
- Accent color: Cyan-Teal (#14A2B9) for secondary interactions, contrast, and data visualization highlights.
- Font pairing: 'Inter' (sans-serif) for functional interface elements and 'Literata' (serif) for formal headings and career reports.
- Minimalist, geometric line icons designed for clarity and alignment with professional scientific and educational standards.
- A structured, modular layout using a progressive stepper for the diagnostic quiz and clean, card-based dashboards for data display.
- Fluid transitions between assessment sections and subtle drawing animations for the intelligence radar charts.
