# CareerCompass Kenya 🇰🇪

**Navigate Your Future with Precision.**

CareerCompass Kenya is a specialized ed-tech platform designed to provide Kenyan students with comprehensive career guidance. By combining modern psychological diagnostics with local educational data, we help students transition from Senior School to tertiary institutions with clarity.

## 🚀 Key Features

- **PIA Assessment**: A deep-dive diagnostic into Passions, Interests, and Abilities.
- **Career Intelligence (MI) Assessment**: Based on Howard Gardner’s 9 Multiple Intelligences.
- **University & TVET Hub**: A directory of top Kenyan institutions (UoN, JKUAT, Strathmore, etc.) with entry requirements.
- **Professional PDF Reports**: Instant, downloadable career roadmaps for students, parents, and counselors.
- **Permanent Dark Theme**: A focused, high-contrast "EdTech" aesthetic.

## 🛠️ Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS & ShadCN UI
- **Backend**: Firebase Authentication & Firestore
- **AI**: Genkit (Google Gemini)
- **Charts**: Recharts
- **PDF Generation**: html2canvas & jsPDF

## 📦 Getting Started

To run the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.
